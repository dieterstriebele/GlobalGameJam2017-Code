- DESIRED OUTPUT ---------------------------------------------------------

Every vertex position/texture coordinate/vertex normal should be written
explicitly into the file. E.g.

f 2/1/1 3/2/1 4/3/1

will be converted to:

v 1.000000 -1.000000 1.000000
vt 1.000000 0.000000
vn 0.000000 -1.000000 0.000000

v -1.000000 -1.000000 1.000000
vt 1.000000 1.000000
vn 0.000000 -1.000000 0.000000

v -1.000000 -1.000000 -1.000000
vt 0.000000 1.000000
vn 0.000000 -1.000000 0.000000

then grouped v/vt/vn are grouped together

v 1.000000 -1.000000 1.000000
v -1.000000 -1.000000 1.000000
v -1.000000 -1.000000 -1.000000

vt 1.000000 0.000000
vt 1.000000 1.000000
vt 0.000000 1.000000

vn 0.000000 -1.000000 0.000000
vn 0.000000 -1.000000 0.000000
vn 0.000000 -1.000000 0.000000

then formatted to java array:

 1.000000f, -1.000000f,  1.000000f,
-1.000000f, -1.000000f,  1.000000f,
-1.000000f, -1.000000f, -1.000000f,

 1.000000f, 0.000000f,
 1.000000f, 1.000000f,
 0.000000f, 1.000000f,

 0.000000f, -1.000000f, 0.000000f,
 0.000000f, -1.000000f, 0.000000f,
 0.000000f, -1.000000f, 0.000000f,

That is done for all "f" index lines! the result are 3 big float arrays,
one for all vertex coordinates, one for all texture coordinates and one for
all normals!

Then everything formatted as java class.

Then the indexes are thrown away!
